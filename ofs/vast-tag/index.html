# Introduction

A VAST tag is a URL that points to a VAST XML feed. The VAST XML feed provides content about the video ads, including media files, companion banners and event tracking pixels. (Please refer to IAB spec https://www.iab.com/guidelines/vast/ (opens new window) for more details). The VAST XML feed is in a standardized format that all video players can understand and use to play video ads.

OperaAds VAST tag points to the VAST XML feed provided by OperaAds ad server. You can load the tag with any video player in your site(or app) to play video ads.

# VAST tag address

https://s.adx.opera.com/tag/ortb/v2/video
