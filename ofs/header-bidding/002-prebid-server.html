# Prebid Server

# Overview

Prebid Server is an open-source solution for server-to-server header bidding. Publisher's client can communicate with Prebid Server to request and receive bids. These bids can then compete directly with bids from publisher's primary ad server. OperaAds has provided bid adapter golang version (opens new window) for prebid server. Prebid server's documentation is available at prebid-server-overview (opens new window), and the source code is available at prebid-server-src (opens new window).

# Request Parameters Example

Following example includes sample imp object with publisherId,endpointId and placementId which can be used to test OperaAds Adapter.

# banner

"imp":[
      {
         "id":"1",
         "banner":{
            "format":[
               {
                  "w":300,
                  "h":250
               }
            ]
         },
         "ext":{
            "operaads":{
               "publisherId":"pub3996476520192",
               "endpointId":"ep3996494184640",
               "placementId":"s4227128550592"
            }
         }
      }
   ]
