# OperaAds Deeplink Integration Guide

# Deeplink

A deep link is a specialized uniform resource identifier (URI) that can open and direct a user to a specific location inside an app and not the home screen. Think of it as the app counterparts of website URLs. They are used to send app users directly to specific in-app locations, thus saving them time and energy finding a particular page themselves. We will add deeplink indicator in a bid request, then advertisers can deside whether to use deeplink in the bid response.

# Specification

# Request

Field Type Description
imp.ext.deeplink int 1 means deeplink supported; 0 means deeplink not supported
imp.ext.fallback int 1 means fallback supported; 0 means fallback not supported

# Response

Field Type Description
seatbid.bid.ext.deeplink string deeplink url to delivery
seatbid.bid.ext.fallback string fallback url to delivery
